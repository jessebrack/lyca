# Link

Primitive for displaying links.

## Standard

<Playground>
    <Link href="#">Hey, I'm a link!</Link>
</Playground>

## Props

| Prop     | Type                     | Required | Description              |
| :------- | :----------------------- | :------- | :----------------------- |
| fontSize | Number, String, or Array | No       | Sets `font-size` of link |
