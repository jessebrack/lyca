# Image

Primitive for displaying images.

## Standard

<Playground>
    <Image src="https://placebear.com/300/300" alt="Bear" />
</Playground>

## Different Sizes

<Playground>
    <Image src="https://placebear.com/100/100" alt="Bear" size="50px" mb={3} />
    <Image src="https://placebear.com/200/200" alt="Bear" size="100px" mb={3} />
    <Image src="https://placebear.com/250/250" alt="Bear" size="150px" />
</Playground>

## Props

| Prop | Type                     | Required | Description                        |
| :--- | :----------------------- | :------- | :--------------------------------- |
| size | Number, String, or Array | No       | Sets `width` and `height` of image |
| m    | Number, String, or Array | No       | Shorthand margin                   |
| mt   | Number, String, or Array | No       | Margin top                         |
| mr   | Number, String, or Array | No       | Margin right                       |
| ml   | Number, String, or Array | No       | Margin left                        |
| mb   | Number, String, or Array | No       | Margin bottom                      |
| mx   | Number, String, or Array | No       | Horizontal margin                  |
| my   | Number, String, or Array | No       | Vertical margin                    |
